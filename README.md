# Фiнальне домашнє завдання курсу

Створи репозиторій **goit-js-hw-08**. Завдання виконуй у файлах `gallery.js` та
`index.html`.  
Прочитай завдання і виконай його в редакторі коду.  
Переконайся, що код відформатований за допомогою `Prettier`,  
а в консолі відсутні помилки і попередження під час  
відкриття живої сторінки завдання.

**Завдання. Галерея зображень**  
Створи галерею з можливістю кліку по її елементах і перегляду  
повнорозмірного зображення в модальному вікні.
