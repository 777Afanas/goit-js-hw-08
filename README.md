# Фiнальне домашнє завдання курсу

Створи репозиторій **goit-js-hw-08**. Завдання виконуй у файлах `gallery.js` та
`index.html`.  
Прочитай завдання і виконай його в редакторі коду.  
Переконайся, що код відформатований за допомогою `Prettier`,  
а в консолі відсутні помилки і попередження під час  
відкриття живої сторінки завдання.

**Завдання. Галерея зображень**  
Створи галерею з можливістю кліку по її елементах і перегляду  
повнорозмірного зображення в модальному вікні.

**На що буде звертати увагу ментор при перевірці:**  
На живій сторінці відображається галерея зображень із  
масиву даних `images`  
Галерея зображень стилізована згідно з макетом  
Дані для галереї створені динамічно в `JS`  
Під час прослуховування події натискання на елементи  
галереї використаний прийом делегування  
При кліку між елементами галереї нічого не відбувається  
Підключена бібліотека `basicLightbox` При кліку по елементу галереї
відкривається модальне  
вікно підключеної бібліотеки, в якому міститься збільшена  
версія зображення, по якому клікнули
